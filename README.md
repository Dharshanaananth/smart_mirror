#  Smart Mirror – Phase 1

## 📌 Overview

Smart Mirror is an interactive dashboard that displays useful information in one place. Users can view weather updates, current date and time, news, motivational quotes, and quickly access Google and YouTube.

This repository contains the completed **Phase-1** implementatio

## ✨ Features

* 🌤 Weather Updates
* 🕒 Date & Time Display
* 📰 News Headlines
* 💡 Motivational Quotes
* 🔍 Google Access
* ▶️ YouTube Access

## 🛠 Technologies Used

* Python
* Web APIs
* Dashboard Interface
* Browser Integration
